# videostady
Перед запуском приложения и сервера, необходимо установить библиотеки npm inslall на сервере и в клиенте.

Подключение:
1) Введите настройки подключения к базе данных в .env
2) запустите файл в папке /videostady/server/migrate.js командой "node server/migrate.js" После завершения процесса миграции базы данных можно запускать приложение и сервер.

Приложение разрабатывалось на тестовом сервере по адресу: 
Клиент: http://help-school66.ru:8080
Сервер: http://help-school66.ru:8084
